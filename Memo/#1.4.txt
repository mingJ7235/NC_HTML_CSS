웹사이트는 2가지의 text ~3가지의 text를 사용한다. 

이제는 text대신 language로 !

웹사이트는 2개~3개의 언어를 사용해서 만들어진다. 
2개거나 3개다

HTML, CSS, JAVASCRIPT

1) HTML -> 이것 쉬운 언어임


2) CSS -> 배우는건 쉽지만 연습해야한다. 


3) Javascript


오직 3가지 언어만 있다. 웹 개발자에게는 3가지만 알면된다! 

웹사이트를 만드는것은 매우 재미있다 그리고 쉽다!
SUPER SIMPLE ! 

--------------------
HTML 이란 무엇인가!?

- Hypertext Markup Language

HTML의 컨셉 : 웹사이트에서 보는 모든 컨텐츠들 (이미지, 텍스트, 타이틀, 링크 등등)
	브라우저는 멍청하다... 브라우저는 인간들이 쓰는 언어를 이해하지 못함 
	브라우저는 우리의 content를 이해 못함. 
	=> 브라우저에게 컨텐트가 무엇인지 알려줘야한다. 
	"브라우저야 이건 title이고 이건 이미지고, 텍스트야" 라고 말해줘야한다. 
	이것들을 설명해주는 것이 HTML
	HTML으로 브라우저에게 contents의 구조를 말해주는 것이다. 

중요한 것은 HTML을 왜, 어떻게 사용하는가!
브라우저에게 웹사이트의 content가 어떻게 구성되어있는지 설명할 때 사용한다. 
오직 HTML만이 할 수 있는 일이다. 
오직 HTML만이 브라우저에게 content의 구조에 대해 말할 수 있다.
