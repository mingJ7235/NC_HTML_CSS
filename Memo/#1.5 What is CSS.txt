Cascading Style Sheet
1) CSS는 HTML과 같이 써야함 should be !!
절대 따로 따로 사용하지 않는다. 두가지를 같이 사용한다. 

2) CSS는 브라우저에게 웹사이트가 '어떻게 보여야하는가에 대해' 알려준다. 
※HTML은 브라우저에게 웹사이트의 content가 무엇인지에 대해 열려주는것.

=>웹사이트의 그 content가 어떻게 보여야하는지 를 알려주는것이 css다

HTML "브라우저야 이건 타이틀이야"
CSS "브라우저야 타이틀은 녹색이야"

HTML "브라우저야 이건 이미지야"
CSS "브라우저야 이 이미지는 가로 25PX 세로 100PX이어야해"

That's it !!
HTML은 뼈대, CSS는 근육 
절대 각각 쓰지 않는다.
HTML이 없다면 CSS는 아무것도 디자인 할 수 없다. 디자인할게 없음. 
HTML만 써서 웹사이트를 만들면 진짜 개 못생김..
